reek

code smell detection for Ruby

What is it

Reek is a tool that will parse your Ruby source code and generate a list of code smells. These warnings will help guide you in refactoring to cleaner code.

Getting started

The first step is to get reek. Install it like this:

gem install reek

And run it like this to scan the current directory you're in:

reek .

You can also check a specific file:

reek demo.rb

Now you can add reek to your rake tasks likewise:

require 'reek/rake/task'

Reek::Rake::Task.new(:reek) do |t|
  t.fail_on_error = false
end

Great. Now you can simply call:

rake reek

Or insert the task into your build process.

You can also add a config.reek file to override default behavior. Here a an example:

---
UncommunicativeVariableName:
  enabled: true
  reject:
  - !ruby/regexp p;^.$/
  - !ruby/regexp /[0-9]$/

There, you're all set!
